# /explore-data - 데이터셋 프로파일링 및 탐색

> 익숙하지 않은 플레이스홀더가 있거나 연결된 도구를 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 참조하세요.

테이블 또는 업로드된 파일에 대한 종합적인 데이터 프로파일을 생성합니다. 분석에 들어가기 전에 데이터의 형태, 품질, 패턴을 먼저 파악합니다.

## 사용법

```
/explore-data <테이블명 또는 파일>
```

## 워크플로우

### 1. 데이터 접근

**데이터 웨어하우스 MCP 서버가 연결된 경우:**

1. 테이블명 확인 (스키마 접두사 처리, 모호한 경우 후보 제안)
2. 테이블 메타데이터 조회: 컬럼명, 타입, 설명(있는 경우)
3. 실제 데이터에 대해 프로파일링 쿼리 실행

**파일이 제공된 경우 (CSV, Excel, Parquet, JSON):**

1. 파일을 읽어 작업 데이터셋으로 로드
2. 데이터로부터 컬럼 타입 추론

**둘 다 없는 경우:**

1. 사용자에게 테이블명(웨어하우스 연결 필요) 또는 파일 업로드를 요청
2. 테이블 스키마를 설명하는 경우, 실행할 프로파일링 쿼리 안내 제공

### 2. 데이터 프로파일 생성

다음 프로파일링 항목을 실행합니다:

**테이블 수준 지표:**
- 전체 행 수
- 컬럼 수 및 타입별 분류
- 대략적인 테이블 크기 (메타데이터에서 확인 가능한 경우)
- 날짜 범위 (날짜 컬럼의 최솟값/최댓값)

**각 컬럼의 컬럼 수준 지표:**
- 데이터 타입 (예상 타입과의 일치 여부 포함)
- Null 개수 및 비율(%)
- 고유값 수 및 카디널리티 (고유값 수 / 전체 행 수)
- 숫자형 컬럼: 최솟값, 최댓값, 평균, 중앙값, 표준편차, 백분위수 (p25, p50, p75, p95, p99)
- 문자열 컬럼: 최소/최대 길이, 최빈값 (상위 10개), 빈 문자열 수
- 날짜/타임스탬프 컬럼: 최솟값, 최댓값, 기간별 분포
- 불리언 컬럼: true/false/null 분포

**컬럼 타입별(차원, 지표, 날짜, ID)로 그룹화한 깔끔한 요약 테이블**로 프로파일을 표시합니다.

### 3. 데이터 품질 이슈 식별

잠재적 문제를 표시합니다:

- **높은 Null 비율**: Null이 5% 초과인 컬럼(경고), 20% 초과인 컬럼(알림)
- **낮은 카디널리티 이상**: 높아야 할 카디널리티가 낮은 컬럼 (예: "user_id"인데 고유값이 50개뿐인 경우)
- **높은 카디널리티 이상**: 범주형이어야 할 컬럼에 고유값이 너무 많은 경우
- **의심스러운 값**: 양수여야 하는데 음수가 있는 경우, 과거 데이터에 미래 날짜가 있는 경우, 명백한 더미값 (예: "N/A", "TBD", "test", "999999")
- **중복 탐지**: 자연키가 있는지, 중복이 있는지 확인
- **분포 왜곡**: 평균에 영향을 줄 수 있는 심하게 치우친 숫자 분포
- **인코딩 문제**: 범주형 필드의 대소문자 혼용, 후행 공백, 일관되지 않은 형식

### 4. 유용한 차원 및 지표 제안

컬럼 프로파일을 바탕으로 추천합니다:

- **최적 차원 컬럼**: 데이터 슬라이싱에 적합한 컬럼 (카디널리티가 적절한 범주형 컬럼, 3~50개 값)
- **핵심 지표 컬럼**: 측정에 유의미한 분포를 가진 숫자형 컬럼
- **시계열 컬럼**: 추세 분석에 적합한 날짜 컬럼
- **자연스러운 그룹화 또는 계층 구조**: 데이터에서 확인된 그룹/계층
- **잠재적 조인 키**: 다른 테이블과 연결 가능한 컬럼 (ID 컬럼, 외래 키)

### 5. 후속 분석 추천

사용자가 다음에 실행할 수 있는 3~5가지 구체적인 분석을 제안합니다:

- "[시간_컬럼] 기준으로 [차원]별 [지표] 추세 분석"
- "[치우친_컬럼] 분포 심층 분석으로 이상값 파악"
- "[문제_컬럼] 데이터 품질 조사"
- "[지표_a]와 [지표_b] 간 상관관계 분석"
- "[날짜_컬럼]과 [상태_컬럼]을 활용한 코호트 분석"

## 출력 형식

```
## 데이터 프로파일: [테이블명]

### 개요
- 행 수: 2,340,891
- 컬럼 수: 23 (차원 8, 지표 6, 날짜 4, ID 5)
- 날짜 범위: 2021-03-15 ~ 2024-01-22

### 컬럼 상세
[요약 테이블]

### 데이터 품질 이슈
[심각도와 함께 표시된 이슈 목록]

### 추천 탐색 분석
[제안된 후속 분석의 번호 목록]
```

## 팁

- 매우 큰 테이블(1억 행 이상)의 경우 프로파일링 쿼리는 기본적으로 샘플링을 사용합니다 — 정확한 수치가 필요하면 말씀해주세요
- 새로운 데이터셋을 처음 탐색할 때, 이 명령으로 구체적인 쿼리를 작성하기 전에 전체적인 현황을 파악할 수 있습니다
- 품질 표시는 휴리스틱 기반입니다 — 모든 표시가 실제 문제는 아니지만, 각각 간단히 확인할 가치가 있습니다
