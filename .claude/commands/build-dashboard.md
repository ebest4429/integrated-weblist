# /build-dashboard - 인터랙티브 대시보드 생성

> 익숙하지 않은 플레이스홀더가 보이거나 연결된 도구를 확인해야 하는 경우 [CONNECTORS.md](../CONNECTORS.md)를 참조하세요.

차트, 필터, 표, 전문적인 스타일을 갖춘 독립형 인터랙티브 HTML 대시보드를 생성합니다. 서버나 외부 의존성 없이 브라우저에서 바로 열 수 있습니다.

## 사용법

```
/build-dashboard <대시보드 설명> [데이터 소스]
```

## 워크플로우

### 1. 대시보드 요구사항 파악

다음 사항을 결정합니다:

- **목적**: 경영진 개요, 운영 모니터링, 심층 분석, 팀 리포팅
- **대상**: 누가 이 대시보드를 사용하는가?
- **핵심 지표**: 가장 중요한 수치는 무엇인가?
- **차원**: 사용자가 필터링하거나 분류할 수 있어야 하는 항목은?
- **데이터 소스**: 라이브 쿼리, 붙여넣기 데이터, CSV 파일, 또는 샘플 데이터

### 2. 데이터 수집

**데이터 웨어하우스가 연결된 경우:**
1. 필요한 데이터를 쿼리합니다
2. 결과를 HTML 파일 내에 JSON으로 임베드합니다

**데이터를 붙여넣거나 업로드한 경우:**
1. 데이터를 파싱하고 정리합니다
2. 대시보드에 JSON으로 임베드합니다

**설명만 있고 데이터가 없는 경우:**
1. 설명된 스키마에 맞는 현실적인 샘플 데이터셋을 생성합니다
2. 대시보드에 샘플 데이터 사용 중임을 표시합니다
3. 실제 데이터로 교체하는 방법을 안내합니다

### 3. 대시보드 레이아웃 설계

표준 대시보드 레이아웃 패턴을 따릅니다:

```
┌──────────────────────────────────────────────────┐
│  대시보드 제목                       [필터 ▼]    │
├────────────┬────────────┬────────────┬───────────┤
│  KPI 카드  │  KPI 카드  │  KPI 카드  │ KPI 카드  │
├────────────┴────────────┼────────────┴───────────┤
│                         │                        │
│    주요 차트             │   보조 차트             │
│    (가장 넓은 영역)      │                        │
│                         │                        │
├─────────────────────────┴────────────────────────┤
│                                                  │
│    상세 표 (정렬 가능, 스크롤 가능)               │
│                                                  │
└──────────────────────────────────────────────────┘
```

**내용에 맞게 레이아웃을 조정합니다:**
- 상단에 2~4개의 KPI 카드로 주요 수치 표시
- 중간 섹션에 1~3개의 차트로 추세 및 분류 표시
- 하단에 드릴다운 데이터를 위한 선택적 상세 표
- 복잡도에 따라 헤더 또는 사이드바에 필터 배치

### 4. HTML 대시보드 구현

다음을 포함하는 단일 독립형 HTML 파일을 생성합니다:

**구조 (HTML):**
- 시맨틱 HTML5 레이아웃
- CSS Grid 또는 Flexbox를 사용한 반응형 그리드
- 필터 컨트롤 (드롭다운, 날짜 선택기, 토글)
- 값과 라벨이 있는 KPI 카드
- 차트 컨테이너
- 정렬 가능한 헤더가 있는 데이터 표

**스타일 (CSS):**
- 전문적인 색상 구성 (깔끔한 흰색, 회색, 데이터용 강조 색상)
- 은은한 그림자가 있는 카드 기반 레이아웃
- 일관된 타이포그래피 (빠른 로딩을 위한 시스템 폰트)
- 다양한 화면 크기에서 작동하는 반응형 디자인
- 인쇄 친화적 스타일

**인터랙티비티 (JavaScript):**
- 인터랙티브 차트를 위한 Chart.js (CDN 포함)
- 모든 차트와 표를 동시에 업데이트하는 필터 드롭다운
- 정렬 가능한 표 컬럼
- 차트 호버 툴팁
- 숫자 형식 지정 (쉼표, 통화, 퍼센트)

**데이터 (임베드 JSON):**
- 모든 데이터를 JavaScript 변수로 HTML에 직접 임베드
- 외부 데이터 요청 불필요
- 완전 오프라인 작동

### 5. 차트 유형 구현

모든 차트에 Chart.js를 사용합니다. 대시보드에서 자주 사용하는 차트 패턴:

- **꺾은선 차트**: 시계열 추세
- **막대 차트**: 카테고리 비교
- **도넛 차트**: 구성 비율 (6개 미만 카테고리)
- **누적 막대 차트**: 시간에 따른 구성 변화
- **혼합형 (막대 + 꺾은선)**: 볼륨과 비율 동시 표시

### 6. 인터랙티비티 추가

**필터:**
```javascript
// 모든 필터가 중앙 필터 상태를 업데이트
// 필터 변경 시 차트와 표가 다시 렌더링됨
function applyFilters() {
    const filtered = data.filter(row => matchesFilters(row));
    updateKPIs(filtered);
    updateCharts(filtered);
    updateTable(filtered);
}
```

**표 정렬:**
- 컬럼 헤더 클릭으로 오름차순/내림차순 정렬
- 현재 정렬 컬럼과 방향을 시각적으로 표시

**툴팁:**
- 차트 호버 시 상세 값 표시
- KPI 카드에 이전 기간 대비 비교값 표시

### 7. 저장 및 열기

1. 대시보드를 설명적인 이름의 HTML 파일로 저장합니다 (예: `sales_dashboard.html`)
2. 사용자의 기본 브라우저에서 엽니다
3. 올바르게 렌더링되는지 확인합니다
4. 데이터 업데이트 또는 커스터마이징 방법을 안내합니다

## 출력 템플릿

생성된 HTML 파일은 다음 구조를 따릅니다:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>[대시보드 제목]</title>
    <script src="https://cdn.jsdelivr.net/npm/chart.js@4.5.1" integrity="sha384-jb8JQMbMoBUzgWatfe6COACi2ljcDdZQ2OxczGA3bGNeWe+6DChMTBJemed7ZnvJ" crossorigin="anonymous"></script>
    <style>
        /* 전문적인 대시보드 CSS */
    </style>
</head>
<body>
    <div class="dashboard">
        <header><!-- 제목 및 필터 --></header>
        <section class="kpis"><!-- KPI 카드 --></section>
        <section class="charts"><!-- 차트 컨테이너 --></section>
        <section class="details"><!-- 데이터 표 --></section>
    </div>
    <script>
        const DATA = [/* 임베드된 JSON 데이터 */];
        // 대시보드 초기화 및 인터랙티비티
    </script>
</body>
</html>
```

## 예시

```
/build-dashboard 월별 매출 대시보드 — 수익 추세, 인기 상품, 지역별 분류 포함. 데이터는 orders 테이블에 있습니다.
```

```
/build-dashboard 지원 티켓 데이터입니다 [CSV 붙여넣기]. 우선순위별 볼륨, 응답 시간 추세, 해결률을 보여주는 대시보드를 만들어주세요.
```

```
/build-dashboard SaaS 회사용 경영진 대시보드 템플릿 — MRR, 이탈률, 신규 고객, NPS 포함. 샘플 데이터 사용.
```

## 팁

- 대시보드는 완전한 독립형 HTML 파일입니다 — 파일을 전송하기만 하면 누구와도 공유 가능합니다
- 실시간 대시보드가 필요하다면 BI 도구 연결을 고려하세요. 이 대시보드는 특정 시점의 스냅샷입니다
- 다른 스타일링을 원하면 "다크 모드" 또는 "프레젠테이션 모드"를 요청하세요
- 브랜드에 맞는 특정 색상 구성을 요청할 수 있습니다
