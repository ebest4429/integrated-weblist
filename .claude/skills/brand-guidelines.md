# Anthropic 브랜드 스타일링

## 개요

이 스킬을 사용하면 Anthropic의 공식 브랜드 아이덴티티 및 스타일 리소스를 적용할 수 있습니다.

**키워드**: 브랜딩, 기업 아이덴티티, 비주얼 아이덴티티, 후처리, 스타일링, 브랜드 색상, 타이포그래피, Anthropic 브랜드, 시각적 포맷, 비주얼 디자인

## 브랜드 가이드라인

### 색상

**주요 색상:**

- 다크: `#141413` - 본문 텍스트 및 어두운 배경
- 라이트: `#faf9f5` - 밝은 배경 및 어두운 배경 위 텍스트
- 미드 그레이: `#b0aea5` - 보조 요소
- 라이트 그레이: `#e8e6dc` - 은은한 배경

**강조 색상:**

- 오렌지: `#d97757` - 1차 강조색
- 블루: `#6a9bcc` - 2차 강조색
- 그린: `#788c5d` - 3차 강조색

### 타이포그래피

- **제목**: Poppins (대체 폰트: Arial)
- **본문**: Lora (대체 폰트: Georgia)
- **참고**: 최적의 결과를 위해 해당 폰트를 미리 환경에 설치해두는 것을 권장합니다

## 기능

### 스마트 폰트 적용

- 제목(24pt 이상)에 Poppins 폰트 적용
- 본문 텍스트에 Lora 폰트 적용
- 커스텀 폰트 미설치 시 Arial/Georgia로 자동 대체
- 모든 시스템에서 가독성 유지

### 텍스트 스타일링

- 제목(24pt 이상): Poppins 폰트
- 본문 텍스트: Lora 폰트
- 배경에 따른 스마트 색상 선택
- 텍스트 계층 구조 및 포맷 유지

### 도형 및 강조 색상

- 텍스트 외 도형에 강조 색상 사용
- 오렌지 → 블루 → 그린 순환 적용
- 브랜드 일관성을 유지하면서 시각적 흥미 확보

## 기술 세부사항

### 폰트 관리

- 시스템에 설치된 Poppins 및 Lora 폰트 우선 사용
- 제목은 Arial, 본문은 Georgia로 자동 대체
- 별도 폰트 설치 불필요 — 기존 시스템 폰트로 동작
- 최적의 결과를 위해 Poppins와 Lora를 환경에 미리 설치 권장

### 색상 적용

- 정확한 브랜드 매칭을 위해 RGB 색상값 사용
- python-pptx의 RGBColor 클래스를 통해 적용
- 시스템 간 색상 충실도 유지
