# 테마 팩토리 스킬

이 스킬은 정교하게 선별된 전문 폰트·색상 테마 컬렉션을 제공합니다. 각 테마는 신중하게 선택된 색상 팔레트와 폰트 조합으로 구성되어 있으며, 선택한 테마를 모든 아티팩트에 적용할 수 있습니다.

## 목적

프레젠테이션 슬라이드 덱 또는 기타 아티팩트에 일관되고 전문적인 스타일을 적용하기 위해 이 스킬을 사용합니다. 각 테마에는 다음이 포함됩니다:
- 헥스 코드가 명시된 통일감 있는 색상 팔레트
- 헤더와 본문용 보완적인 폰트 조합
- 다양한 맥락과 청중에 적합한 고유한 시각적 정체성

## 사용 방법

슬라이드 덱 또는 다른 아티팩트에 스타일을 적용하려면:

1. **테마 쇼케이스 표시**: `theme-showcase.pdf` 파일을 표시하여 사용자가 모든 테마를 시각적으로 확인할 수 있도록 합니다. 파일을 수정하지 말고 보기용으로만 표시합니다.
2. **선택 요청**: 덱에 적용할 테마를 선택해 달라고 요청합니다.
3. **선택 대기**: 선택한 테마에 대한 명확한 확인을 받습니다.
4. **테마 적용**: 테마가 선택되면 해당 테마의 색상과 폰트를 덱/아티팩트에 적용합니다.

## 사용 가능한 테마

다음 10가지 테마를 사용할 수 있으며, 각각 `theme-showcase.pdf`에서 확인 가능합니다:

1. **Ocean Depths (오션 뎁스)** - 전문적이고 차분한 해양 테마
2. **Sunset Boulevard (선셋 불러바드)** - 따뜻하고 생동감 있는 노을 색상
3. **Forest Canopy (포레스트 캐노피)** - 자연적이고 안정적인 대지의 색조
4. **Modern Minimalist (모던 미니멀리스트)** - 깔끔하고 현대적인 그레이스케일
5. **Golden Hour (골든 아워)** - 풍부하고 따뜻한 가을 팔레트
6. **Arctic Frost (아틱 프로스트)** - 시원하고 청명한 겨울 느낌 테마
7. **Desert Rose (데저트 로즈)** - 부드럽고 세련된 먼지빛 색조
8. **Tech Innovation (테크 이노베이션)** - 대담하고 현대적인 기술 미학
9. **Botanical Garden (보태니컬 가든)** - 신선하고 유기적인 정원 색상
10. **Midnight Galaxy (미드나이트 갤럭시)** - 극적이고 우주적인 깊은 색조

## 테마 상세 정보

각 테마는 `themes/` 디렉토리에 다음 내용이 포함된 완전한 사양으로 정의되어 있습니다:
- 헥스 코드가 명시된 통일감 있는 색상 팔레트
- 헤더와 본문용 보완적인 폰트 조합
- 다양한 맥락과 청중에 적합한 고유한 시각적 정체성

## 적용 프로세스

원하는 테마가 선택되면:
1. `themes/` 디렉토리에서 해당 테마 파일을 읽습니다.
2. 명시된 색상과 폰트를 덱 전체에 일관되게 적용합니다.
3. 적절한 대비와 가독성을 확보합니다.
4. 모든 슬라이드에서 테마의 시각적 정체성을 유지합니다.

## 커스텀 테마 생성

기존 테마 중 아티팩트에 맞는 것이 없을 경우 커스텀 테마를 생성합니다. 제공된 입력을 바탕으로 기존 테마와 유사한 새 테마를 생성합니다. 폰트/색상 조합을 잘 나타내는 유사한 이름을 붙여줍니다. 제공된 기본 설명을 바탕으로 적절한 색상/폰트를 선택합니다. 테마를 생성한 후 검토 및 확인을 위해 보여줍니다. 그 다음 위에 설명된 대로 테마를 적용합니다.
